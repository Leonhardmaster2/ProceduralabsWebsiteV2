Getting Started

Welcome to Proceduralabs documentation. This guide will help you get up and running with our UE5 plugins and assets.

Installation

All our products can be installed through the Epic Games Launcher:

  1. Open the Epic Games Launcher
  2. Navigate to the Unreal Engine > Library tab
  3. Find your purchased product in the Vault
  4. Click "Install to Engine" and select your UE5 version

First Steps

After installation, enable the plugin in your project:

1. Edit > Plugins
2. Search for the plugin name
3. Check the "Enabled" checkbox
4. Restart the editor
